# FortniteAssist - Assistive Technology for Fortnite Mobile

## 🧭 Mission Statement
To empower **blind, visually impaired, and physically disabled individuals** to play **Fortnite Mobile** independently through real-time on-device AI vision that detects enemies, weapons, and critical game elements — then provides **adaptive aim guidance** via simulated input using Android's Accessibility Service — all while ensuring **user privacy, ethical integrity, performance efficiency, and future adaptability**.

**This is NOT a cheat tool. It is an assistive technology designed to level the playing field for players who cannot see or operate standard controls due to disability.**

## ⚖️ Core Principles
- **Accessibility First**: UI/UX fully navigable via TalkBack, switch control, voice navigation
- **Privacy by Design**: Zero off-device data transmission without explicit consent
- **Ethical Boundaries**: Strictly an assistive tool, not a cheat engine
- **Performance Optimized**: Runs smoothly on mid-tier devices
- **Maintainable Architecture**: Modular, testable, well-documented codebase

## 🛠️ Technical Stack
- **Android SDK (API 34+)**
- **React Native (0.73+)** for accessible UI
- **Kotlin** for core logic and native modules
- **TensorFlow Lite** for on-device AI inference
- **MediaProjection API** for screen capture
- **AccessibilityService** for input simulation
